# عالم بدر Native Android

هذه نسخة أندرويد أصلية Native Flutter وليست WebView.

## المزايا
- واجهة Flutter أصلية احترافية.
- نفس روح عالم بدر: بدر الدب، بطاقات كبيرة، ألعاب، قصص، إنجازات، لوحة الوالدين.
- فصل واضح بين العربي والإنجليزي.
- زر صوت عربي يستخدم ar-JO ويقرأ الكلمة مع الشرح.
- زر English Sound يستخدم en-US ويقرأ الإنجليزية فقط بتكنيك مختلف.
- بناء APK مباشر عبر GitHub Actions داخل apk/badr.apk.
