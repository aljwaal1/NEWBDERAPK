# عالم بدر للتعلم والمرح - APK Android

نسخة Flutter Android شبيهة بفكرة تطبيق الويب، لكنها تطبيق APK حقيقي للأندرويد.

## المحتوى
- شاشة رئيسية مبهرة وبسيطة للأطفال.
- تعلم الحروف والأرقام والحيوانات والألوان والأشكال والمواصلات.
- قصص قصيرة للأطفال.
- ألعاب تعليمية بسيطة.
- صفحة إنجازات.
- صفحة الوالدين.

## البناء
يتم بناء APK تلقائيًا عبر GitHub Actions، والنتيجة توضع في:

`apk/badr.apk`

بدون Release وبدون AAB وبدون مكتبات إضافية.
